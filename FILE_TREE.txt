KEY FILE PURPOSES:
==================

CONFIGURATION FILES:
- package.json: All dependencies and npm scripts
- next.config.js: API proxy configuration for backend integration
- tailwind.config.js: Custom blue gradient theme and animations
- tsconfig.json: TypeScript configuration with path aliases
- .env.local.example: Environment variables template

CORE APPLICATION:
- app/layout.tsx: Root layout with providers
- app/page.tsx: Home page redirect
- components/Providers.tsx: React Query and state providers
- lib/api.ts: Axios client with JWT interceptors
- lib/auth.ts: Authentication state management

BUSINESS LOGIC:
- hooks/: Custom hooks for all data operations
- lib/constants.ts: All API endpoints and routes
- types/: Complete TypeScript definitions
- lib/mockData.ts: Development mock data

UI COMPONENTS:
- components/Layout/: Navigation and layout
- components/[Feature]/: Feature-specific components
- styles/globals.css: Tailwind component classes

PAGES (App Router):
- app/dashboard/: Main user dashboard
- app/auth/: Authentication pages
- app/profile/: Profile management
- app/internships/: Internship browsing
- app/projects/: Project management
- app/skillgenie/: AI learning assistant

DEPENDENCIES SUMMARY:
====================

PRODUCTION:
- Next.js 14: React framework with App Router
- React 18: UI library
- TypeScript: Type safety
- Tailwind CSS: Utility-first styling
- React Query: Server state management
- Zustand: Client state management
- React Hook Form: Form handling
- Zod: Schema validation
- Axios: HTTP client
- Recharts: Data visualization
- Lucide React: Icon library

DEVELOPMENT:
- ESLint: Code linting
- Prettier: Code formatting
- TypeScript ESLint: TypeScript linting
- Tailwind Prettier Plugin: Class sorting

INTEGRATION POINTS:
==================

BACKEND API ENDPOINTS:
- Authentication: /auth/signup, /auth/signin, /auth/refresh
- Profile: /profile, /profile/upload-resume
- Skills: /skills, /skills/gaps
- Internships: /internships, /internships/classify
- Projects: /projects, /copilot/generate-project
- Validation: /validate/submit, /validate/{id}

ENVIRONMENT VARIABLES:
- NEXT_PUBLIC_API_URL: Backend API URL
- NEXT_PUBLIC_USE_MOCK_API: Mock/real API toggle
- NEXT_PUBLIC_JWT_SECRET: JWT secret key
- File upload configuration variables

This structure provides a complete, production-ready frontend application
with comprehensive documentation for seamless backend integration.
